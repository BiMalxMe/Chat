# SapiansChat

SapiansChat is a real-time chat application built with Node.js, Express, and Socket.io. It allows users to create accounts, join chat rooms, and send messages to other users in real-time.

## Features

- User authentication with JWTs
- User registration and profile management
- Chat room creation and joining
- Real-time messaging between users
- Message persistence with MongoDB
- Image uploading and storage with Cloudinary
- Voice message support (user-to-user and group chats)
- Full-screen image viewer for chat images
- **Multiplayer Quiz Battle** - Challenge your friends to a trivia quiz battle! Answer questions faster than your opponent to score points!
- Responsive design

## Getting Started

1. Clone the repository: `git clone https://github.com/bimalchalise/chat.git`
2. Install dependencies: `npm install`
3. Create a `.env` file in the root directory and add your environment variables (see `.env.example` for reference)
4. Start the application: `npm run start`
